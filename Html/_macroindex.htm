Urban Institute SAS Macro Library

Macro Description
Adjust_count_totals Adjusts a list of integer count variables against a control total.
Capitalize Autocall macro to capitalize the first letter of a text string.
Compare_file_struct Compares the file structure of two or more data sets.
Compile_num_desc Compiles numeric statistics (n, sum, mean, std, min, max) for all variables in a data set. Default is to put statistics in a data set called WORK._DESC_&ds_name.
Create_metadata_html Creates HTML pages from metadata repository created by the Update_metadata() macro.
Dataset_exists Returns TRUE (1) if a data set exists, FALSE (0) if not.
Data_to_format Create a format using values from a data set
Delete_metadata_file Deletes metadata for a SAS data set.
Delete_metadata_library Deletes a library from the metadata system.
DirExist Returns 1 if specified folder exists, 0 if it does not exist. Code source is http://www.sascommunity.org/wiki/Tips:Check_if_a_directory_exists
Dissimilarity_index Autocall macro to calculate the dissimilarity index between two populations.
Dollar_convert Converts dollars using CPI.
DSLibOnly Autocall macro that returns the library name portion of a libname.dataset specification.
DSNameOnly Autocall macro that returns the data set name portion of a libname.dataset specification.
Dup_check Check data set for duplicate obs. by key variables and print out duplicates.
Err_mput Write a macro-generated error message to the SAS LOG using %PUT.
Err_put Write a macro-generated error message to the SAS LOG using PUT.
Fdate Autocall macro to put a formatted version of a date value into a macro variable.
File_info Autocall macro to print basic information for a data set: contents, first few obs., and descriptive statistics.
FmtSearch Autocall macro to manage multiple SAS format libraries. Written by Pete Lund, presented at SUGI 28, "Keep Those Formats Rolling: A Macro to Manage the FMTSEARCH= Option." <http://www2.sas.com/proceedings/sugi28/116-28.pdf>
Freq_table SAS autocall macro to create a data set containing frequency distributions for a set of variables from an input data set. Each value of a variable in the input data set is a single obs in the output data set. The output data set contains the following variables: Variable - The name of the variable Value - The raw value of the variable (nb: this is a char var) FmtValue - The formatted value of the variable. If variable is unformatted, then this is the same as Value. Frequency - Frequency of the value for the variable Percent - Percentage of value for the variable CumFrequency - Cumm. frequency of the value for the variable CumPercent - Cumm. percentage of value for the variable
GetProgDrive Autocall macro to return the drive letter of the currently submitted SAS program to the global macro variable given by VAR. If running in interactive mode, macro returns blank.
GetProgName Autocall macro to return the name of the currently submitted SAS program to the global macro variable given by VAR. If running in interactive mode, macro returns "(Interactive)".
ListChangeDelim Change delimiters in a macro variable list: default is change from spaces to commas. Optionally add prefix to list items.
ListDelete Autocall macro to remove entries from a list of items.
ListIntersect Autocall macro returns the intersection of two lists.
ListNoDup Autocall macro returns list with duplicate entries removed.
ListUnion Autocall macro returns a union of items between two lists. All duplicate items are removed.
MacroSearch Autocall macro to manage multiple SAS macro libraries. Adapted from FmtSearch by Pete Lund.
MCapitalize Autocall macro returns macro text value with first letter capitalized and the rest lowercase.
Moe_conv Autocall macro that returns a recalculated margin of error based on a new confidence level. Method based on http://www.census.gov/acs/www/Downloads/handbooks/ACSResearch.pdf p. A-12.
Moe_prop Autocall macro that returns a calculation for a margin of error for a derived proportion based on the values and margins of error of the proportion numerator and denominator. The numerator of a proportion must be a subset of the denominator (e.g., the proportion of single person households that are female). Method based on http://www.census.gov/acs/www/Downloads/handbooks/ACSResearch.pdf pp. A-14 - A-15.
Moe_prop_a Autocall macro that returns a calculation for a margin of error for a derived proportion based on the values and margins of error of the proportion numerator and denominator. The numerator of a proportion must be a subset of the denominator (e.g., the proportion of single person households that are female). Alternate version of macro that assigns value to a specified variable (var=), rather than just returning the MOE value as %Moe_prop() does. This version tests whether quantity in sqrt() is negative and, if so, uses the %Moe_ratio() value instead, as suggested in the Census doc. Assigned value can be scaled by optional multiplier (mult=1). Method based on http://www.census.gov/acs/www/Downloads/handbooks/ACSResearch.pdf pp. A-14 - A-15.
Moe_ratio Autocall macro that returns a calculation for a margin of error for a derived ratio based on the values and margins of error of the proportion numerator and denominator. The numerator of a ratio is NOT a subset of the denominator (e.g., the ratio of females living alone to males living alone).
Moe_sum Autocall macro that returns a calculation for a margin of error for a derived sum of values from the individual margins of error comprising the sum.
Moving_avg Calculate a moving average for variables of the form xxx_nnn and assign to specified variable. Label variable.
mparam_is_no Returns 1 if the macro parameter value is "No" (could be "n", "N", "nO", etc.)
mparam_is_yes Returns 1 if the macro parameter value is "Yes" (could be "y", "Y", "Yes", "YE", etc.)
Note_mput Write a macro-generated note message to the SAS LOG using %PUT.
Note_put Write a macro-generated note message to the SAS LOG using PUT.
Pop_option Pop specified SAS system option from top of stack and restore as current option value.
Push_option Push specified SAS system option unto top of stack for later recovery through %Pop_option macro.
SAS_compatibility Submits system options for backward compatibility of earlier SAS versions. Currently supports backward compatibility of versions 9.3 and 9.4 to 9.2.
Sort_array_ref autocall macro to 'sort' the elements in an array by reference. Given an array, A{*}, the macro creates a temporary array called A_SRTD{*} that contains a list of indices for the original array that will put its elements in sorted order.
Super_transpose Autocall macro that performs transpose on a series of vars in a data set and combines results together into a single file.
Transform_geo_data Autocall macro to convert data from an original geographic level to new geography using a normalized weighting file.
Update_metadata_file Registers metadata for a SAS data set.
Update_metadata_library Registers metadata for a SAS library.
Warn_mput Write a macro-generated warning message to the SAS LOG using %PUT.
Warn_put Write a macro-generated warning message to the SAS LOG using PUT.
